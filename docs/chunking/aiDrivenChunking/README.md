# AI-Driven Chunking

Purpose: Uses an LLM (Databricks Chat) to produce semantically coherent chunks. A mock implementation is provided for local testing.

Usage:

```bash
python3 chunking/ai_driven_chunking.py
```

Notes:
- The mock function `perform_ai_driven_chunking_mock` is safe for local use.
- When using a real LLM, configure the Databricks endpoint and credentials.
