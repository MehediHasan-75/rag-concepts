# Recursive / Code Chunking

Purpose: Language-aware recursive chunking for code and technical documents. Preserves functions, classes, and imports as metadata for better retrieval.

Usage:

```bash
python3 chunking/recursive_chunking.py
```

Notes:
- Example input file: `docs/chunking/recursive_chunking/python_code.md`.
- Output is written to `docs/chunking/recursive_chunking/chunk_output.txt`.
